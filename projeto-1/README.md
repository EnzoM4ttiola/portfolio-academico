## 📄 Descrição
Este é um sistema básico de cadastro desenvolvido como parte das atividades do curso de Desenvolvimento Web.

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript

## 💻 Como Executar
1. Clone o repositório
2. Navegue até a pasta `projeto-1`
3. Abra o arquivo `index.html` no navegador
